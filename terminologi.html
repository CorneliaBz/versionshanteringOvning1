Repository

Projektets mapp som innehåller projektfilerna. Förvarar även varje fils ändringar och historik.

Branch

En parallel version av repository. Används för att kunna göra ändringar och tester utan att "förstöra" live-versionen.

Clone

En lokal kopia av hela projektet som sparas på din dator. Används för att man ska kunna jobba offline utan att behöva vara uppkopplad eller förlita sig på en server.
